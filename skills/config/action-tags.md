# Action Tags

Used by `curator-filter` to assign actionable labels to 01/02-grade articles.

| Tag | Meaning | Example |
|-----|---------|---------|
| `[TRY_TOOL]` | Tool/product worth testing | "Scrapling could replace agent-browser fallback" |
| `[CONTENT_MATERIAL]` | Reusable quotes, cases, or data points | "Pieter Levels quote can be adapted for BIP tweet" |
| `[REPLICATE_STRATEGY]` | Method/framework worth reproducing | "12 startups in 12 months methodology" |
| `[NETWORK_LEAD]` | Person or community worth reaching out to | "Author runs a quant Discord" |
| `[ARCHIVE_ONLY]` | No action needed, store as reference | Reference data, background reading |
| `[NEEDS_HUMAN]` | Curator cannot determine, escalate to human | Unreliable source but relevant content |
